aspcol.movingmicrophone.spherical_jn

aspcol.movingmicrophone.spherical_jn(v, z)

Calculates j_v(z), the spherical Bessel function of the first kind of order v

Currently limited to orders 0, 1 and 2. See documentation for spherical_jn_max_order_2 for more information.

Parameters:

  • v (ndarray of shape (num_points,)) – order of the spherical Bessel function

  • z (ndarray of shape (num_points,)) – argument of the spherical Bessel function

Returns:

order_v – spherical Bessel function of the first kind of order v

Return type:

ndarray of shape (num_points,)
