aspcol.plot.soundfield_estimation_comparison(pos_est, p_est, p_true, freqs, fig_folder, shape='', center=None, num_ls=1, output_method='pdf', pos_mic=None, images=None, image_true=None, pos_image=None, remove_freqs_below=0, remove_freqs_above=inf, num_examples=4, points_for_errors=None)ÔÉÅ

Outputs a series of plots comparing a set of sound field estimates to the true sound field.

The function accepts sound field estimates in the frequency domain.

Parameters:

  • pos_est (np.ndarray) ‚Äì Positions of the estimated sound field. Shape (num_positions, spatial_dim)

  • p_est (np.ndarray) ‚Äì Estimated sound field. Shape (num_freqs, num_ls, num_positions) or (num_freqs, num_positions)

  • p_true (np.ndarray) ‚Äì True sound field. Shape (num_freqs, num_ls, num_positions) or (num_freqs, num_positions)

  • freqs (np.ndarray of shape (num_freqs,)) ‚Äì Frequencies used in the simulation.

  • fig_folder (pathlib Path) ‚Äì Folder to save the figures in.

  • shape (str, optional) ‚Äì Shape of the evaluation point array. If any of the valid options ‚Äúrectangle‚Äù, ‚Äúcircle‚Äù is used, then a few more plots will be created. The default is ‚Äú‚Äù.

  • center (np.ndarray, optional) ‚Äì Center of the circle if shape is ‚Äúcircle‚Äù. The default is None.

  • num_ls (int, optional) ‚Äì Number of loudspeakers. The default is 1.

  • output_method (str, optional) ‚Äì Method for saving the plot. Options are ‚Äòshow‚Äô, ‚Äòtikz‚Äô, ‚Äòpdf‚Äô, ‚Äòsvg‚Äô, ‚Äònone‚Äô. If ‚Äòshow‚Äô, the plot is shown in a window. If ‚Äònone‚Äô, the plot is not saved. The default is ‚Äúpdf‚Äù.

  • pos_mic (np.ndarray, optional) ‚Äì Positions of the microphones. Shape (num_mics, spatial_dim) The positions are added to the sound field images if supplied. The default is None.

  • points_for_errors (np.ndarray of shape (num_positions,) with Boolean values, optional) ‚Äì Points which have True values are included in error calculations. The default is None. It also specifies which points to use in setting of colormap limits in the soundfield MSE plots. Useful if a few points are known exactly, which usually leads to useless sound field MSE plots.
