aspcol.soundfieldestimation

Collection of algorithms for sound field estimation

  • Kernel interpolation [uenoKernel2018]

  • Infinite dimensional spherical harmonic analysis for moving microphones [brunnstromBayesianSubmitted]

  • Spatial spectrum estimation for moving microphones [katzbergSpherical2021]

References

Saruwatari, “Kernel ridge regression with constraint of Helmholtz equation for sound field interpolation,” in 2018 16th International Workshop on Acoustic Signal Enhancement (IWAENC), Tokyo, Japan: IEEE, Sep. 2018, pp. 436–440. doi: 10.1109/IWAENC.2018.8521334. [link]

[brunnstromBayesianSubmitted] J. Brunnström, M. B. Møller, and M. Moonen, “Bayesian sound field estimation using moving microphones,” IEEE Open Journal of Signal Processing, submitted.

Mertins, “Spherical harmonic representation for dynamic sound-field measurements,” in ICASSP 2021 - 2021 IEEE International Conference on Acoustics, Speech and Signal Processing (ICASSP), Jun. 2021, pp. 426–430. doi: 10.1109/ICASSP39728.2021.9413708. [link]

Functions

est_inf_dimensional_shd_dynamic(p, pos, ...)

Estimates the RIR at evaluation positions using data from a moving microphone using Bayesian inference of an infinite sequence of spherical harmonics

est_ki_diffuse(p, seq, pos, pos_eval, ...)

Estimates the RIR in the frequency domain using kernel interpolation Assumes seq is a perfect periodic sequence

est_ki_diffuse_freq(p_freq, pos, pos_eval, ...)

Estimates the RIR in the frequency domain using kernel interpolation Uses the frequency domain sound pressure as input

est_spatial_spectrum_dynamic(p, pos, ...[, ...])

Estimates the RIR at evaluation positions using data from a moving microphone

pseq_nearest_neighbour(p, seq, pos, pos_eval)

Estimates the sound field at the evaluation points by simply selecting the value associated with the nearest microphone position.

shd_min_order(wavenumber, radius)

Returns the minimum order of the spherical harmonics that should be used for a given wavenumber and radius

shd_num_degrees(max_order)

Returns a list of mode indices for each order when order = n, the degrees are only non-zero for -n <= degree <= n

shd_num_degrees_vector(max_order)

Constructs a vector with the index of each order and degree when order = n, the degrees are only non-zero for -n <= degree <= n
