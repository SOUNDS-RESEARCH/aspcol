Audio Signal Processing Collection (aspcol)

This is a package for audio signal processing. Contains implementations of state-of-the-art sound field estimation and sound field reproduction methods, as well as some basic audio signal processing tools.

API Reference

aspcol.adaptivefilter

Implements several basic adaptive filters.

aspcol.correlation

Implements autocorrelation and cross-correlation estimators, as well as related useful calculations.

aspcol.distance

Calculates distance measures for some different types of quantities.

aspcol.filterclasses

Implementations of various types of filters

aspcol.filterdesign

Functions to design and compute filters

aspcol.kernelinterpolation

Interpolation of a sound field taking physical properties of sound into account.

aspcol.lowrank

Implements algorithms related to low-rank tensor approximations of impulse responses

aspcol.matrices

Functions for common operations on matrices.

aspcol.montecarlo

Functions for Monte Carlo integration

aspcol.pseq

Functions associated with perfect periodic sequences

aspcol.soundfieldcontrol

Algorithms for sound field control, in particular sound zone control.

aspcol.soundfieldestimation

Collection of algorithms for sound field estimation

aspcol.utilities

Helper functions for various tasks without a clear category
